# Gbox 开发任务清单

## 当前目标

完成第一版单页网页，让用户可以点击盒子召唤角色，角色在页面中运动，点击角色进入对应游戏。

## 后续素材任务

- [x] 建立角色与游戏入口表。
- [x] 确认每个角色对应的具体游戏。
- [ ] 确认每个角色的最终跳转链接。
- [x] 准备或生成角色图片素材。
- [x] 将占位符号替换为角色图片。
- [x] 根据最终素材调整角色尺寸和运动效果。

## 精灵动画任务

- [x] 根据 `game.txt` 建立精灵图制作计划。
- [ ] 确认首批要做的 5 个精灵角色。
- [ ] 准备首批 sprite sheet。
- [ ] 新建 `assets/sprites/` 素材目录。
- [ ] 将角色数据升级为 sprite 配置。
- [ ] 用 CSS `steps()` 播放精灵帧动画。
- [ ] 根据角色类型实现走路、跳跃、飞行或冲刺运动。
- [ ] 点击角色时切换点击反馈动作。
